# Hydration Reminder

Periodic reminders to drink water throughout the day.

## Usage

```bash
dotnet run --project HydrationReminder.csproj
```

## Reminder Intervals

| Option | Interval |
|--------|----------|
| 1 | Every 30 minutes |
| 2 | Every hour |
| 3 | Every 2 hours |
| 4 | Custom |

## Features

- **Customizable intervals** - Set your preferred reminder frequency
- **Countdown display** - See time until next reminder
- **Daily tracking** - Tracks reminders remaining
- **Audio notification** - Console beep for attention

## Health Benefits

- Improved energy levels
- Better concentration
- Healthy skin
- Proper digestion
- Temperature regulation
