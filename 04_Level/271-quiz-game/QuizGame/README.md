# Quiz Game

A multiple-choice quiz game with scoring and feedback.

## Usage

```bash
dotnet run --project QuizGame.csproj
```

## Example

```
🧠 Quiz Game
============

Question 1/10:
What does CPU stand for?

  1. Central Processing Unit
  2. Central Power Unit
  3. Computer Personal Unit
  4. Central Process Utility

Your answer (1-4): 1
✅ Correct!

...

========================================

📊 Final Score: 8/10 (80%)
🌟 Great job!
```

## Concepts Demonstrated

- Arrays and collections
- User input validation
- Score tracking
- Conditional feedback
- Class design
