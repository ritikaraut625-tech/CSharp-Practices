# Mood Tracker

Track daily mood and identify emotional patterns over time.

## Usage

```bash
dotnet run --project MoodTracker.csproj
```

## Mood Scale

| Rating | Emoji | Description |
|--------|-------|-------------|
| 5 | 😄 | Excellent |
| 4 | 🙂 | Good |
| 3 | 😐 | Neutral |
| 2 | 🙁 | Bad |
| 1 | 😞 | Terrible |

## Features

- **Daily logging** - Quick mood entry with notes
- **History view** - See recent entries
- **Pattern analysis** - Average mood, best/worst days
- **Weekly breakdown** - Mood by day of week

## Concepts Demonstrated

- JSON serialization
- LINQ grouping and aggregation
- Statistics calculation
- Emoji mapping
