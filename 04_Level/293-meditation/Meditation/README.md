# Meditation Timer

Guided meditation timer with preset durations and bell sounds.

## Usage

```bash
dotnet run --project Meditation.csproj
```

## Meditation Types

| Type | Duration |
|------|----------|
| Quick Calm | 3 minutes |
| Mindfulness | 10 minutes |
| Deep Meditation | 20 minutes |
| Custom | Your choice |

## Features

- **Preset durations** - Quick selection of common times
- **Countdown display** - Real-time remaining time
- **Bell sounds** - Opening and closing bells
- **Interval bells** - Gentle reminder every 5 minutes

## Concepts Demonstrated

- TimeSpan handling
- Console timing
- Audio feedback (Console.Beep)
- User input handling
