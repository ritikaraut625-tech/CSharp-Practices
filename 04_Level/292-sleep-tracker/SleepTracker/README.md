# Sleep Tracker

Track sleep patterns, quality, and duration with statistics.

## Usage

```bash
dotnet run --project SleepTracker.csproj
```

## Features

- **Sleep logging** - Record bedtime, wake time, and quality
- **History view** - See recent sleep entries
- **Statistics** - Average hours and quality
- **Quality rating** - 1-5 star rating system

## Concepts Demonstrated

- JSON serialization
- TimeSpan handling
- Statistics calculation
- Data aggregation
