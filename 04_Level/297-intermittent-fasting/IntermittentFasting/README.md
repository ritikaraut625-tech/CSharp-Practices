# Intermittent Fasting Tracker

Track fasting sessions with real-time progress and statistics.

## Usage

```bash
dotnet run --project IntermittentFasting.csproj
```

## Features

- **Start/End tracking** - Log fasting sessions
- **Real-time progress** - See elapsed and remaining time
- **Target setting** - Customize fasting goals (16:8, 18:6, etc.)
- **History** - View past sessions
- **Statistics** - Average duration, longest fast, completion rate

## Common Fasting Protocols

| Protocol | Fast | Eating Window |
|----------|------|---------------|
| 16:8 | 16 hours | 8 hours |
| 18:6 | 18 hours | 6 hours |
| 20:4 | 20 hours | 4 hours |
| OMAD | 23 hours | 1 hour |

## Concepts Demonstrated

- DateTime calculations
- Progress visualization
- JSON persistence
- Statistics aggregation
