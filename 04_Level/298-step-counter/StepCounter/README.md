# Step Counter

Track daily steps and monitor progress toward fitness goals.

## Usage

```bash
dotnet run --project StepCounter.csproj
```

## Features

- **Step logging** - Add steps manually or from fitness tracker
- **Daily goal** - Set and track step targets
- **Progress visualization** - Visual progress bar
- **7-day history** - View recent activity
- **Statistics** - Averages, best days, goal completion

## Common Step Goals

| Goal Level | Daily Steps |
|------------|-------------|
| Sedentary | < 5,000 |
| Active | 10,000 |
| Highly Active | 15,000+ |

## Concepts Demonstrated

- JSON serialization
- Date-based grouping
- Progress visualization
- Statistics calculation
