# Breathing Exercise

Guided breathing exercises with customizable patterns.

## Usage

```bash
dotnet run --project Breathing.csproj
```

## Breathing Patterns

| Pattern | Inhale | Hold | Exhale | Hold |
|---------|--------|------|--------|------|
| Box Breathing | 4s | 4s | 4s | 4s |
| 4-7-8 Relaxation | 4s | 7s | 8s | - |
| Coherent | 5s | - | 5s | - |
| Custom | Your choice |

## Features

- **Preset patterns** - Popular breathing techniques
- **Custom patterns** - Create your own rhythm
- **Visual countdown** - Follow along on screen
- **Cycle tracking** - See current cycle number

## Benefits

- **Box Breathing** - Stress reduction, focus
- **4-7-8** - Relaxation, sleep aid
- **Coherent** - Heart rate variability, calm

## Concepts Demonstrated

- Timing loops
- Console animation
- User input handling
- Pattern customization
