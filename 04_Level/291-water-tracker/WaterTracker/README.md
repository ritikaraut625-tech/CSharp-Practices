# Water Intake Tracker

Track daily water intake with visual progress and goal setting.

## Usage

```bash
dotnet run --project WaterTracker.csproj
```

## Features

- **Daily goal tracking** - Set and track hydration goals
- **Quick logging** - Log standard 250ml servings
- **Custom amounts** - Log any amount
- **Visual progress** - Progress bars for intake
- **7-day history** - View weekly hydration patterns

## Concepts Demonstrated

- JSON serialization
- Date-based filtering
- Progress visualization
- Data persistence
